The report creation system allows citizens to document infrastructure problems with precise location data, detailed descriptions, and photographic evidence.

## Accessing the Report Form

After logging in, navigate to the "Crear Reporte" (Create Report) section to begin documenting an infrastructure issue.

## Step-by-Step Report Creation

<Steps>
  <Step title="Set the location">
    Choose how to specify where the problem is located:
    
    **Option 1: Use your current location**
    - Click the "📍 Usar mi ubicación" (Use my location) button
    - Grant location permissions when prompted
    - The map will center on your current position
    
    **Option 2: Click on the map**
    - Navigate the interactive map to find the problem location
    - Click directly on the map where the issue exists
    
    The system automatically:
    - Captures latitude and longitude coordinates
    - Generates a geohash for precise location indexing
    - Retrieves the address using reverse geocoding
    
    <Tip>
      You can zoom in/out on the map using your mouse wheel or the +/- controls for more precise location marking.
    </Tip>
  </Step>
  
  <Step title="Classify the problem">
    Use the cascading dropdowns to categorize the issue:
    
    1. **Problemática** (Problem Category) - Select the main problem type
    2. **Clase de problemática** (Problem Class) - Choose the specific class (enabled after step 1)
    3. **Tipo de problema** (Problem Type) - Select the exact type (enabled after step 2)
    
    <Note>
      Each dropdown automatically populates based on your previous selection, ensuring accurate categorization.
    </Note>
  </Step>
  
  <Step title="Specify the location details">
    Provide geographic context:
    
    1. **Departamento** (Department) - Select the department from the dropdown
    2. **Ciudad** (City) - Select the city (automatically filtered by department)
    3. **Dirección** (Address) - Review and edit the auto-filled address if needed
  </Step>
  
  <Step title="Describe the problem">
    In the "Descripción del problema" text area:
    - Provide a clear description of the issue
    - Include relevant details about the problem's impact
    - Stay within the 250 character limit
    
    <Warning>
      Be specific and factual in your description. This helps agents understand and address the issue more effectively.
    </Warning>
  </Step>
  
  <Step title="Upload 6 photos">
    Visual evidence is **required** for all reports. You must upload exactly 6 photos:
    
    1. Click on any of the 6 image boxes
    2. Select an image file from your device
    3. The preview will appear in the box
    4. Repeat for all 6 photo slots
    
    <Tip>
      Capture different angles and aspects of the problem:
      - Wide shots showing the overall area
      - Close-ups of specific damage
      - Context shots showing nearby landmarks
      - Before/after comparisons if applicable
    </Tip>
    
    **Photo Requirements:**
    - Format: JPG, JPEG, or PNG
    - All 6 slots must be filled
    - Images should clearly show the problem
  </Step>
  
  <Step title="Submit the report">
    Once all fields are complete:
    
    1. Review all information for accuracy
    2. Click the "Crear reporte" (Create report) button
    3. Wait for the upload to complete
    4. You'll see a success message when finished
    
    The system will:
    - Validate all required fields
    - Upload all 6 photos to secure storage
    - Create a new case or associate with an existing case
    - Send you back to the reports dashboard
  </Step>
</Steps>

## What Happens Next

After you submit your report:

1. **Case Grouping** - The system checks if similar reports exist at the same location
   - If yes: Your report is added to an existing case
   - If no: A new case is created

2. **Photo Storage** - Your 6 photos are saved with unique identifiers

3. **Initial Status** - Your report starts with "Pendiente" (Pending) status

4. **Agent Assignment** - Government agents can now view and respond to the case

<Note>
  Multiple reports about the same location are grouped into a single "case" to help agents address systemic issues efficiently.
</Note>

## Tips for Effective Reports

<Tip>
  **Be Precise with Location**
  - Zoom in closely before clicking on the map
  - Verify the address matches the actual location
  - Use "Usar mi ubicación" only when you're physically at the problem site
</Tip>

<Tip>
  **Take Quality Photos**
  - Use good lighting
  - Hold your device steady
  - Include reference points (street signs, buildings)
  - Show the scale of the problem
</Tip>

<Tip>
  **Write Clear Descriptions**
  - State what the problem is immediately
  - Mention any safety hazards
  - Note how long the problem has existed
  - Avoid opinions; stick to facts
</Tip>
