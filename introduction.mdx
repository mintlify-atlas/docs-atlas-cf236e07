# Welcome to Quejas Ciudadanas

Quejas Ciudadanas is a comprehensive citizen complaint management platform designed to streamline the process of reporting and tracking infrastructure issues. Built with ASP.NET Core MVC, it provides a powerful solution for governments and municipalities to engage with citizens and manage public service requests efficiently.

## What is Quejas Ciudadanas?

Quejas Ciudadanas ("Citizen Complaints" in English) is a full-featured SaaS platform that enables:

- **Citizens** to report infrastructure problems with precise geolocation, detailed descriptions, and photo evidence
- **Government Agents** to review, categorize, and respond to citizen reports with status updates and documentation
- **Administrators** to track case metrics, identify problem hotspots, and manage resolution workflows

## How It Works

<Steps>
  <Step title="Citizen Registration">
    Citizens create an account with their identification document and basic information
  </Step>
  <Step title="Report Creation">
    Citizens report infrastructure problems with location, photos, and descriptions
  </Step>
  <Step title="Automatic Case Grouping">
    The system groups similar reports by location and problem type into cases
  </Step>
  <Step title="Agent Review">
    Government agents review cases, prioritize them, and investigate issues
  </Step>
  <Step title="Response & Resolution">
    Agents respond with status updates, photos, and documentation until resolution
  </Step>
</Steps>

## Use Cases

Quejas Ciudadanas is ideal for:

- **Municipal Governments** managing public infrastructure complaints
- **Utilities Companies** tracking service issues and outages
- **Transportation Authorities** monitoring road conditions and signage problems
- **Public Works Departments** coordinating maintenance and repairs
- **Environmental Agencies** tracking pollution and sanitation issues

## Technology Stack

- **Framework**: ASP.NET Core 8.0 MVC
- **Database**: Microsoft SQL Server with Entity Framework Core
- **Authentication**: Session-based authentication with custom filters
- **Geolocation**: Geohash-based clustering for case grouping
- **File Storage**: File system storage for photos and documents
